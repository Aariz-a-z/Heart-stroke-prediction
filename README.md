# Heart Stroke Prediction System

A machine learning-based application that predicts the risk of heart stroke using patient health data. The model analyzes various medical attributes to assist in early detection and preventive decision-making.

## Features
- Data preprocessing and cleaning
- Feature selection and transformation
- Logistic Regression-based classification
- Model evaluation and accuracy analysis
- Predictive insights for stroke risk

## Methodology
- Collected and analyzed healthcare dataset
- Handled missing values and performed data preprocessing
- Applied feature scaling for better model performance
- Trained a Logistic Regression model for classification
- Evaluated model using accuracy, precision, and recall

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Output
- Prediction of stroke risk (High / Low)
- Model performance metrics
- Visualizations of important features
